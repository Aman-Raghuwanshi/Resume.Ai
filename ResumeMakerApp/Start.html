About The Project

This project was built as part of the PapyrusNebulae Hackathon organized by Adobe. The objective of the project was to develop a resume generation application using the Adobe Pdfservices-node-sdk. The project was developed by Aman Raghuwanshi, a student at IIT Guwahati.
